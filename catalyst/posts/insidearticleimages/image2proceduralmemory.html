Optimize Your Study with Memory Science

Apply Dr. Michael T. Ullman’s Declarative/Procedural Model to tailor your learning techniques— engage the right memory system for each discipline.

Language

+

Declarative Memory

  • Spaced repetition of new vocabulary to strengthen hippocampal consolidation.
  • Active retrieval practice (flash-cards, self-quizzing) for durable recall.
  • Multimodal encoding: pair words with images, example sentences, or audio.

Procedural Memory

  • Immersive practice—speak, listen, think in the target language daily.
  • Interleaved grammar drills (mix patterns rather than block one rule at a time).
  • Prioritize sleep after rule practice to enhance offline procedural consolidation.

Mathematics

+

Declarative Memory

  • Drill arithmetic facts (e.g. times-tables) with spaced flash-cards.
  • Self-testing on formulas and definitions instead of passive review.
  • Create concept maps linking related theorems and formulas.

Procedural Memory

  • Work through step-by-step worked examples to internalize algorithms.
  • Use interleaved problem sets across topics (algebra, geometry, etc.).
  • Overlearn key procedures until they become automatic.

Music

+

Declarative Memory

  • Memorize short motifs with repeated listening and notation drills.
  • Quiz yourself on scales, chords, and terminology using flash-cards.
  • Link musical patterns to visual or narrative mnemonics.

Procedural Memory

  • Practice passages slowly to build accurate muscle memory.
  • Use a metronome for consistent rhythmic drilling.
  • Maintain daily, focused repetition of technical exercises.
