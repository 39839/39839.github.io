## Important Notes

### Image URLs
You need to replace the placeholder image URLs with your actual hosted images:

1. **Header Image:** Replace `https://raw.githubusercontent.com/yourusername/repo/main/NewsletterHeader1.png`
   - Upload `NewsletterHeader1.png` to your GitHub repo or image hosting service
   - Get the direct URL and replace in the template

2. **Logo Image:** Replace `https://raw.githubusercontent.com/yourusername/repo/main/NewGlassLogo.png`
   - Upload `NewGlassLogo.png` to your GitHub repo
   - Get the direct URL and replace in the template

3. **Bottom Image:** Replace `https://raw.githubusercontent.com/yourusername/repo/main/3DLogo2.jpg`
   - Upload `3DLogo2.jpg` to your GitHub repo
   - Get the direct URL and replace in the template

### Quick Image Hosting Options:

**Option 1: GitHub (Recommended)**
1. Go to your GitHub repo
2. Create an `images` folder
3. Upload: `NewsletterHeader1.png`, `NewGlassLogo.png`, `3DLogo2.jpg`
4. Get URLs like: `https://raw.githubusercontent.com/YOUR-USERNAME/CatalystMagazine/main/NewsletterHeader1.png`

**Option 2: ImgBB or Imgur**
1. Go to https://imgbb.com/ or https://imgur.com/
2. Upload your images
3. Get direct image URLs
4. Replace in template

**Option 3: Your GitHub Pages Site**
1. Images are already in your repo root
2. Use URLs like: `https://YOUR-USERNAME.github.io/CatalystMagazine/NewsletterHeader1.png`

## Setup Steps

### Step 1: Host Your Images
1. Choose a hosting method above
2. Upload these 3 images:
   - `NewsletterHeader1.png`
   - `NewGlassLogo.png`
   - `3DLogo2.jpg`
3. Get the direct URLs

### Step 2: Update Template
1. Copy the HTML above
2. Replace all image URLs with your actual URLs
3. Update social media links if needed

### Step 3: Add to EmailJS
1. Go to https://dashboard.emailjs.com/admin/templates
2. Click "Welcome" template (`template_4846tjj`)
3. Paste the HTML into the template editor
4. Save

### Step 4: Test
1. Go to your website
2. Subscribe with a test email
3. Check inbox for the beautiful email!
